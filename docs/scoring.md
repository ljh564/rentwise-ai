# 推荐评分规则

当前评分为确定性规则，不使用 LLM，也不代表概率。

```text
成本分：最高 35
通勤分：最高 35
偏好命中：每项 +5
硬条件全部通过：+20
存在冲突：条件分降为 max(0, 8 - 冲突数 × 2)
公平性惩罚：多成员最大与最小通勤差 × 0.15
```

成本分：

```text
max(0, 35 - max(0, 月均成本 - 综合预算 × 75%) / 120)
```

通勤分：

```text
max(0, 35 - 加权单程通勤分钟 × 0.5)
```

硬条件包括租金、综合成本、卧室、面积、出租方式、电梯、宠物、中介费、区域和每个目的地的通勤上限。

自定义偏好当前只与房源结构化标签精确匹配。LLM 接入后会增加开放文本偏好解析，但最终加分依据仍应作为结构化证据返回。
